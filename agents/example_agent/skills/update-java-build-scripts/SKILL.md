---

When updating build scripts:

## Modifications
- Update Maven/Gradle build files for compatibility with the target framework
- Modify compiler settings, plugin configurations, and build phases
- Ensure build scripts reference correct framework versions and repositories

## Validation
- Execute a test compilation to ensure build scripts are correct
- Document build script changes in `CHANGELOG.md`
